# Fase de Control y Cierre del Proyecto — HorarioSmart

Esta carpeta contiene la documentación formal correspondiente a la **fase de control y cierre** del proyecto, elaborada conforme a la consigna del curso *Taller de Proyectos 2 — Ingeniería de Sistemas e Informática*, alineada al enfoque **PMBOK** y a las buenas prácticas de ingeniería de software orientadas a calidad, trazabilidad y mantenibilidad.

Todos los documentos se presentan en formato **Markdown** y forman parte del repositorio versionado del proyecto.
